# Part 4 - Tableau Executive Dashboard & Data Storytelling

## Business Problem

The objective of this project is to design an executive Tableau dashboard that enables retail leadership to monitor overall business performance and support strategic decision-making through interactive visualizations and business storytelling.

---

## Dataset Description

The dataset contains retail transaction records including:

- Orders
- Sales
- Profit
- Discounts
- Customer Segments
- Product Categories
- Regional Information
- Shipping Details
- Return Information

---

## Tableau Workbook

File:

```
tableau/executive_dashboard.twbx
```

The workbook contains multiple analytical worksheets and one executive dashboard.

---

## Calculated Fields

The following calculated fields were created:

- Profit Margin
- Cost
- Average Order Value
- Return Rate
- Shipping Delay Bucket

---

## Dashboard Components

The dashboard includes:

- Total Sales KPI
- Total Profit KPI
- Profit Margin KPI
- Sales Trend
- Regional Performance
- Category Profitability
- Discount vs Profit Analysis

---

## Filters and Interactions

Interactive dashboard features include:

- Region Filter
- Category Filter
- Dashboard Filter Actions

These allow users to dynamically explore business performance across multiple dimensions.

---

## Key Business Insights

- Total Sales exceeded ₹217 million.
- Profit Margin remained above 15%.
- South region achieved the highest sales.
- Technology products generated the highest profit.
- Higher discounts generally reduced profitability.
- Sales remained relatively stable throughout the analysis period.
- Low-performing product categories require strategic review.
- Dashboard identifies multiple opportunities for regional and product expansion.

---

## Dashboard Story

The dashboard provides an executive-level overview of business performance, highlighting growth trends, regional contributions, product profitability, and discount effectiveness. The visualizations collectively support strategic planning and operational decision-making.

---

## Assumptions and Limitations

### Assumptions

- Dataset is complete and accurate.
- Financial values are represented consistently.
- Calculated fields correctly represent business metrics.

### Limitations

- Dashboard uses historical data only.
- Inventory, customer lifetime value, and forecasting are not included.
